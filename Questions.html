• 1.6 Describing access issues that need taking into account - describe issues that people with disabilities may face when using webpages and how we can overcome them Web accessibility encompasses all disabilities that affect access to the Web, including:
-auditory -cognitive -neurological -physical -speech -visual
To work around these issues, many people use assistive technologies to browse the internet. This includes screen readers that vocalize the text on each page, speech recognition software that converts speech into text, Braille terminals, and even alternative keyboards that accommodate special needs.
I have not included any accessibility features into the site but I understand that if I had more time, I could potentially include some of the afore mentioned factors to facilitate users with disabilities.
References: https://www.w3.org/WAI/test-evaluate/preliminary/ https://www.dreamhost.com/blog/make-your-website-accessible/
